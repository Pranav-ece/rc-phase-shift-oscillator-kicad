
Description
This file demonstrates the design and simulation of an RC Phase Shift Oscillator using KiCad. 
Theory
An RC Phase Shift Oscillator works on the principle of phase shift:
- Each RC stage provides 60° phase shift
- Total from 3 stages = 180°
- Op-amp provides additional 180°
- Total phase shift = 360° → sustained oscillations

Tools Used
- KiCad (Schematic design)

 Learning Outcome
- Understood oscillator design principles
- Learned feedback and phase shift conditions

Future Improvements
